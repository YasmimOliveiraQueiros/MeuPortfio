Meu portfólio é um projeto pessoal desenvolvido com HTML, CSS e JavaScript, criado para apresentar de forma clara e organizada quem eu sou, minhas habilidades, projetos, certificados e formas de contato.

O site foi pensado para ser responsivo, acessível e visualmente agradável, funcionando bem em diferentes dispositivos. Além disso, possui interações que tornam a navegação mais dinâmica, como troca de temas, filtro de projetos, menu mobile, botão para copiar e-mail e visualização ampliada da foto de perfil.

A proposta do projeto é reunir, em um só lugar, informações importantes sobre minha trajetória e meu interesse pela área de tecnologia, com foco em desenvolvimento de software, inteligência artificial e banco de dados.

Esse portfólio também representa minha evolução prática no desenvolvimento web, unindo organização, identidade visual e uma apresentação mais profissional dos meus conhecimentos e projetos.
